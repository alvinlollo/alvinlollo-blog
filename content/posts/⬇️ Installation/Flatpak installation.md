## Flatpak

#### Ubuntu Flatpak

Add repos

```bash
sudo add-apt-repository ppa:flatpak/stable
```

#### Raspbian Flatpak 

Raspbian isn't special just run the installation using ``apt``
```bash
sudo apt install flatpak
```
